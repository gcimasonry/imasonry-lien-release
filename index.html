☁️ Google Drive Sync

Sync your project database to a shared Google Drive file so all office staff see the same data in real time.
1

Sign in with Google

Each person signs in with their own Gmail account. The file owner must share the Drive file with coworkers (edit access).
2

Connect the shared project file

The file owner clicks "Create New Drive File" below to create imasonry-projects.json in their Drive, then shares it with coworkers. Everyone else pastes the File ID from the shared link.
3

Sync

Once connected, the app pulls from Drive on startup and pushes every time you save a project. You can also manually sync anytime.

Before You Print

For a clean single-page output, disable browser headers & footers:

  • Chrome / Edge: More settings → Headers and footers → off
  • Safari: Print → uncheck "Print headers and footers"
  • Firefox: Page Setup → Margins & Header/Footer → set all to --blank--
